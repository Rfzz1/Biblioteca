Bem vindo ao meu **Sistema de Biblioteca**, um programa em Python que permite o usuário acessar funcionalidades específicas de retirada e disponibilidade de livros

---
#Funcionalidades

1. **Login de Administrador**
   - O administrador do sistema possui um acesso exclusivo (admin, Y90R$Lpi!)
   - Ele pode cadastrar funcionários, apenas
   
2. **Login Funcionário**
   - Todos os funcionários cadastrados pelo administrador podem fazer o login atráves do acesso: (PrimeiroNomeFuncionário, PrimeiroNomeFuncionário@2025Biblioteca)
   - O funcionáro poderá cadastrar um novo livro, receber devolução de livro ou remover um livro da biblioteca

3. **Cadastro de Leitor**
   - O leitor poderá fazer seu próprio cadastro
   
4. **Login Leitor**
   - O leitor fará o login, com as mesmas informações do cadastro (CPF)
   - Poderá retirar o livro à sua escolha, se estiver cadastrado na biblioteca
   
5. **GUI**
   - O sistema vem junto de uma interface gráfica que tem as mesmas funcionalidades citadas acima
